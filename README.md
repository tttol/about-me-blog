# about-me-blog
This is my blog site. <br/>
https://blog.about-tttol.link/

## Commands

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build |
| `npm run new "Title"` | Create a new blog post |

### Create a new post
```bash
npm run new "My New Article"
```
This creates `src/content/blog/my-new-article.md` with frontmatter (title, pubDate) auto-filled.
